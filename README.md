# アプリ名: shift_manage
> アプリ名は、プロジェクトの顔でもあるので、適切な命名を行ってください。

概要:
> 概要には、このアプリで何ができるのかを簡潔に書きましょう。

シフトの希望日を管理するシステム

ユーザー、シフト希望日（日付時間帯）、シフト希望を提出した時間の登録を行い、スタッフごとのシフト希望提出回数(今月)
時間帯ごとの人気倍率(今月)男女比率をグラフなどで表示する簡易アプリケーション

## アピールポイント
この部分に、発表に替わる内容を書きます。
アプリケーション動作のサンプル動画などを貼り付けられると良いです。
※動画の貼り付けは、GIFアニメーションなどでも可です。

スタッフの男女比率、今月のシフト希望においての人気の時間帯とスタッフがどれだけシフト希望を提出したかをグラフなどで表示し、一目でわかりやすくしました。
